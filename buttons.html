Button styles

Bootstrap has the following classes:

  • .btn
  • .btn-default
  • -btn-primary
  • .btn-success
  • .btn-info
  • .btn-warning
  • .btn-danger
  • .btn-link

The button classes can be used on these tags: a, button, or input.

Button Sizes

Bootstrap provides four button sizes. The classes that define the different sizes are:

  • .btn-lg
  • .btn-sm
  • .btn-xs
